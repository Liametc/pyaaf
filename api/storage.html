save()¶

Save AAF file to disk. If not path and the mode is ‘rw’ or ‘w’ it will overwrite or modify the current file. If path is supplied a new file will be created, (Save Copy As). If the extension of the path is .xml a xml file will be saved. Note: If file mode is ‘t’ or ‘r’ and path is None, nothing will happen
